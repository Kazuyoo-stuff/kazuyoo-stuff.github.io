# Changelogs
### [v2.0]
- Complete rewrite from prop file to full shell script framework with GPL license
- Added multi-GPU support for Adreno, Mali, and PowerVR drivers
- Fixed optimize_gpu_frequency v2 — volt extraction variable from freq to gpu_freq which was undefined
- Fixed optimize_gpu_temperature — threshold check raised from < 90000 to < 95000 to match written value
- Added dynamic GPU frequency scaling with thermal limit bypass
- Implemented task cgroup and nice priority optimizations for GPU/rendering processes
- Added memory management tuning (swappiness, VFS cache, dirty writeback)
- Integrated FPSGO kernel optimizations with gpu_block_boost and rescue enablement
- Added hardware-specific path detection for gpufreq, gpufreqv2, and GED parameters
- Implemented Vulkan/OpenGL renderer auto-detection for driver configuration
- Added game list integration with automatic driver opt-in/opt-out for Android 12+
- Created helper functions for task renice and cgroup assignments
- Added file masking system to lock critical GPU parameters
- Implemented comprehensive logging for all parameter changes
- Set GPU devfreq governor to performance mode with disabled idle timers
- Reduced memory readahead to 128KB for faster I/O response
- Added SurfaceFlinger GL backpressure disabling
- Enabled MSM touch boost and power manager touch boost
- Disabled fsync and vsync at kernel module level for reduced latency
